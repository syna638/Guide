Key Terms You'll Learn

Here are some important terms you'll encounter throughout this guide. Don't worry if they seem confusing now ‚Äì we'll explain each one in detail!

Term Simple Explanation
Frontend The part of the website you see and interact with (buttons, forms, colors)
Backend The hidden part that processes data and talks to the database
API A messenger that carries data between frontend and backend
Database Where all the information is stored (like complaints, users)
Middleware Security guard that checks every request before it's processed
Authentication Proving who you are (like showing an ID)
PWA A website that can be installed like an app on your phone
